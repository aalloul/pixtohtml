DirtyForms How-To

Example usage

All forms in the page will be observed Whenever an input value changes it will get the ‘changed’ class added to it

$(function(){
  $("form").dirty_form();
});

// All forms in the page will be observed
// Whenever an input value changes it will get the 'forever_changes' class added to it
$(function(){
  $("form").dirty_form({changedClass: "forever_changes"});
});

Often times you want to apply the “changed” CSS class to more than one element. To support this, DirtyForm can take a “addClassOn” option. Pass in a function that will be executed in the context of the dirty input element and the return value of the function will be added to the list of elements that get the “changed” class. In this function “this” refers to the dirty element. The code below will add the “change” class to the dirty input and to any labels descendants of the li element that contains the input. Example:

$("form").dirty_form({
  addClassOn: function(){ 
    return this.parents("li").find("label");
  }
});
