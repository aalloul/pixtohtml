Mojolicious::Plugin::TtRenderer

Template Renderer Plugin for Mojolicious

SYNOPSIS

:

plugin 'tt_renderer';



$self->plugin('tt_renderer');

DESCRIPTION

This plugin is a simple Template Toolkit renderer for . Its defaults are usually reasonable, although for finer grain detail in configuration you may want to use directly.
