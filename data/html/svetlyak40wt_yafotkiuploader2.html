./yaploader albums

Она выдаст примерно список альбомов:

1) "Умолчательный альбом", 21 image(s)
2) "Screenshots", empty
3) "Пейзажи", 16 image(s)

Далее, можно воспользоваться командой upload:

./yaploader upload -a 3 forest.jpg sunset.jpg

В этом примере, мы загружаем два снимка в альбом “Пейзажи”. Обратите внимание, что при этом указывается его номер. Номер альбома непостоянен, и может измениться при удалении какого-либо другого альбома. Так что перед загрузкой полезно запросить список альбомов и уточнить номер.

Многие команды позволяют задавать дополнительные параметры. Чтобы узнать, какие именно, пользуйтесь встроенной справкой: ./yaploader --help или ./yaploader имя-команды --help.

Настройки по-умолчанию

Некоторые настройки “по-умолчанию” можно хранить в конфиге. Для этого создайте в домашней директории файл .fotki.conf. Конфиг может содержать такие опции:

# доступ к фоточке может тах же быть friends или private
access_type = public
# закрыть доступ к фотографии по URL со страниц вне домена Яндекс.Фоток
storage_private = no
# отключить комментарии
disable_comments = no
# только для взрослых
xxx = no
# не показывать оригинал
hide_original = no
