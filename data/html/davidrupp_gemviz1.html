gemviz

DESCRIPTION:

Graph the dependencies of an installed gem. Uses a similar approach to DepGraph, but finer-grained; graphs dependencies only for the named gem(s), rather than for all installed gems at once.

FEATURES/PROBLEMS:

  • Doesn’t take multiple versions into account.
  • [TODO] Add ability to graph reverse-dependencies.

SYNOPSIS:

gemviz rails

REQUIREMENTS:

  • ruby-graphviz

INSTALL:

  • gem install gemviz
