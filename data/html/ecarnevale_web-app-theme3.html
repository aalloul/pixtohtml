Contributing

  • Fork this repository.
  • Duplicate the ‘themes/default’ folder and rename it.
  • Modify the style.css file adding your favorite colors.
  • Add a link to your theme in the ‘Switch Theme’ block inside the index.html file.
  • Send a pull request.
