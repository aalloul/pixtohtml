XUI

A lightweight, dead simple, microtiny, super modular JavaScript framework for building mobile web applications.

XUI Application Authors and Hackers

Its true: the minified code is super tiny. You can find more information, downloads and documentation at http://xuijs.com â€”if you want to build your own custom XUI or help contribute: please read on!

XUI Contributors and Hackers

To build xui: run rake in the shell of your choice from the root of the project directory. (This requires Ruby.) There are other tasks for code minification, running the specs and generating docs. Run rake -T to see them all.
