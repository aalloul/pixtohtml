require ‘rodf’

ss = RODF::Spreadsheet.new t = ss.table ‘My first table from Ruby’ r = t.row c = r.cell ‘Hello, rODF world!’

two methods to write to file

ss.write_to ‘my-spreadsheet.ods’ # or File.write(‘my-spreadsheet.ods’) do |f| f.write ss.bytes # you can send your data in Rails over HTTP using the bytes method end ```

Both styles can be mixed and matched at will:

require 'rodf'

ss = RODF::Spreadsheet.new
ss.table 'My first table from Ruby' do
  row do
    cell 'Hello, rODF world!'
  end
end

ss.write_to 'my-spreadsheet.ods'
