Why

By releasing this code under a liberal MIT open source license, we hope to empower other people so they can better organize and participate in more events that support free sharing of information, open society, and involved citizenry.

Releases

The stable releases this software are tagged with version numbers, such as “v0.20090416”, that represent the date they were made. There is also a “stable” branch that points to the current stable release. The file describes significant changes made between stable releases.
