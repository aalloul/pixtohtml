Domain Operations

```ruby # Return the domain name d.name # => ‘example.com’

Check if domain is locked

d.locked? # => true

Check if domain is unlocked

d.unlocked? # => false

Unlock the domain for transfer

d.unlock

Lock the domain to the registrar

d.lock

Get name servers

d.nameservers # => [‘ns1.example.com’, ‘ns2.example.com’, ‘ns3.example.com’]

Update name servers

d.update_nameservers([‘ns1.example.com’, ‘ns2.example.com’, ‘ns3.example.com’])

Get expiration date

d.expiration_date # => Returns a Date object

Check if domain is expired

d.expired? # => true

Get registration status

d.registration_status # => ‘Registered’ or ‘Expired’

Renew domain (defaults to 1 year)

d.renew!
