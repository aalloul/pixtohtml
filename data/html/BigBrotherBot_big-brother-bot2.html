For developers

B3 is an extremely extensible software. Support for new online games can be added quite easily by creating new parsers (if the game server meets the necessary requirements), and B3 functionalities can be enhanced by means of plugins. You can browse B3 code reference : it will give you a general idea on the structure of B3 core and B3 plugins. You can also find some plugins and parsers examples in the example directory.
