Netzke Basepack

A pack of pre-built components that can be used as building blocks for your webapps.

Included components

Basepack includes the following components:

    • a grid panel with a thick bag of features
    • a tree panel with features similar to the Grid
    • a form panel with automatic binding of fields
    • a window which stores its size, position, and maximized state
    • a full-window component usually used as one-page application base

Besides, Basepack implements:

  • persistence of region sizes and collapsed states of an arbitrary component that uses Ext’s border layout (see
    • a plugin that allows enhancing any grid with live search functionality

Requirements

  • Ruby >= 1.9.3
  • Rails ~> 4.2.0
  • Ext JS = 5.1.1

Installation

In your Gemfile:

gem 'netzke-basepack'

For the “edge” stuff, instruct bundler to get the gem straight from GitHub:

gem 'netzke-basepack', github: "netzke/netzke-basepack"
