Math::GSL

Math::GSL is a Perl interface to the . The GNU Scientific Library (GSL) is a numerical library for C and C++ programmers. It is free software under the GNU General Public License. Math::GSL uses SWIG to generate Perl bindings to most GSL functionality.

Dependencies

CPAN distribution dependencies

Currently Math::GSL requires at least Perl 5.8.1 and GSL 1.15 to compile. It also requires that the gsl-config binary can be found in your PATH or to find GSL.

Git repo dependencies

SWIG 2.x is needed to build Math::GSL from the git repo, in addition to all the dependencies above. 2.0.8 or newer is required to work with Perl 5.20 and higher.

Installation

To install this module, run the following commands:

perl Build.PL
./Build
./Build test
./Build install clean
