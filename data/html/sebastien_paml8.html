In case you want to add additional class attributes or specific attributes, you can add (and override) the included elements attributes by doing:

%include widget/datepicker +.Imported(date=2009-10-23)

in the above example, the ‘widget/datepicker.paml’ snippet will be included, the ‘Imported’ class will be added to the element class attributes and a ‘date’ attribute will be set with the value ‘2009-10-23’.
