Building

A proper internet connection is sufficient to build FitNesse. The build process will bootstrap itself by downloading and from there will download the dependencies required to build and test FitNesse.

To build and run all tests, run the command

$ ./gradlew

which builds the all target.

NB. On windows call gradlew.bat instead of ./gradlew.
