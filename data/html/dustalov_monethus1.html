Allcash

Nice and Simple Personal Finance Manager

Allcash (form. Monethus) is a pretty simple personal finance manager, whitten in Ruby and Shoes.

Dependencies

  • Ruby 1.9.1
  • Shoes 3.0.0 (Policeman)
  • Sequel 3.16.0
  • SQLite/Ruby 1.3.1

When first started, Allcash will download and install all necessary gems automatically (via Shoes.setup).

I hope that above versions of the dependencies should work fine, too.

Running

UNIX®-like Operating Systems

Ensure that shoes executable is in PATH, then just run ./allcash from his directory.

In case when shoes isn’t in PATH, please tune the SHOES_PATH variable in the allcash start script.

Microsoft® Windows®

Currently, not.
