Install

Our public Git repository is hosted on GitHub and can be viewed at

https://github.com/oscurrency/oscurrency

You can install a new deployment on Heroku with the command:

$ rake heroku:install

Update a Heroku deployment with:

$ rake heroku:update

Administration and Configuration

After install, log in with the default admin@example.com account (password is admin) and click on the “Admin View” link in the upper right. This will take you to the Site administration page (. Click on “Preferences” in the left panel and your app name will be listed. Click on the pencil icon to edit preferences.
