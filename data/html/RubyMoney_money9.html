# or number: currency: format: delimiter: “,” separator: “.” ```

For this example Money.new(123456789, "SEK").format will return 1,234,567.89 kr which otherwise will return 1 234 567,89 kr.

If you wish to disable this feature:

Money.use_i18n = false

Troubleshooting

If you get a runtime error such as:

I18n::InvalidLocale: :en is not a valid locale

Set the following:

I18n.enforce_available_locales = false

Formatting

There are several formatting rules for when Money#format is called. For more information, check out the .

If you wish to format money according to the EU’s in either English, Irish, Latvian or Maltese:

m = Money.new('123', :gbp) # => #<Money fractional:123 currency:GBP>
m.format( symbol: m.currency.to_s + ' ') # => "GBP 1.23"
