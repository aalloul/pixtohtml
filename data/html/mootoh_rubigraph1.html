Rubigraph

A Ruby wrapper for dynamic visualization library, . It provides a shortcut to plot, draw a graph (by calling XML-RPC internally).

See the to get complete description about API.

Example

Make sure Ubigraph serverr is started before using this library.

require 'rubigraph'
Rubigraph.init

v1  = Vertex.new
v2  = Vertex.new
e12 = Edge.new(v1, v2)

v1.color  = '#003366'
v2.shape  = 'sphere'
e12.label = 'edge between 1 and 2'

Install

Rubigraph is included in the official package. Download Ubigraph and install it.

If you ever want it with your modifications, simply type:

% gem install rubigraph
