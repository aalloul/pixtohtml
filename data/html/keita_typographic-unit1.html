typographic-unit

typographic-unit is a Ruby library for converting between typographic units according to TeXâ€™s way. This library can handle the following units:

  • TeX Scaled Point(sp)
  • TeX Point(pt)
  • Pica(pt)
  • Inch(in)
  • TeX Big Point(bp)
  • PostScript Point(ps_pt)
  • Meter(m)
  • Centimeter(cm)
  • Milimeter(mm)
  • Didot Point(dd)
  • Cicero(cc)
  • Japanese Q(q)
  • American Point(american_pt)
  • JIS Point(jis_pt)
  • Japanese Gou(gou)

Installing

gem install typographic-unit

How to use

Basics

require "typographic-unit"
1.cm          # => #<1cm>
1.cm - 1.mm   # => #<0.9cm>
1.cm == 10.mm # => true
1.in >> :cm          # => #<2.54cm>
2.54.cm >> :in       # => #<1.0in>
1.in - 1.cm          # => #<0.606299212598425in>
(1.in - 1.cm) >> :cm # => #<1.54cm>

Convert
