## Installation ## Basics for installing SIMS locally (without the tests, which don’t work on Windows yet)

git clone git://github.com/vegantech/sims.git
cd sims

Make sure you’re running version 1.9.3 of Ruby and run :

rvm install ruby-1.9.3-p484
