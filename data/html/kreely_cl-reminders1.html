Common Lisp Reminders

This module does basically what remind by David Skoll does. However, as a library, it allows for more options and expansions than remind (in my humble opinion).

I’ve chosen to write this library in Common Lisp to try doing something vaguely useful in CL and so that other CL apps can pull it in if there is not something like this already (NOTE: Found one set of date utilities somewhat similar to this library on CLIKI or some other CL site).

Remind (the command line utility) is a useful tool, but only allows creation of text reminders (for calendars or for echoing to the command line) or commands that can be run on a given date (via the RUN keyword). To do more than this with a reminder requires entering a reminder twice. CL-Remind will not need to have reminders entered twice.

The aim of this utility is to allow a user to simply create a number of reminders and then do with them as they wish.

The things you might want to do with a reminder are (not an exhaustive list)

  • Create an ICS file from a reminder to send to someone or to use in software that uses ICS files (NOTE: Need to make a read/write ICS module).
  • Create a marked calendar from a group of reminder entries.
  • Apply an algorithm to existing entries to work out some sort of ‘how hard am I working ?’ index.
  • Send email warning of reminders due to occur.
  • …And generally allow a richer scripting environment for managing reminder based data.

Some Examples

A simple reminder for Saint Pats day is

(rem-text "Saint Patrick's day" :day 17 :month 3)

If there are no date params this means the reminder is triggered every day

(rem-text "Learn something new today")

Reminders can have repeats

(rem-text "Green rubbish bin goes out" :d 24 :m May :repeat-days 14)

Reminders can have repeats with an allowed number of repeat counts.

(rem-text "Remind 3 times" :d 8 :m Jan :repeat-days 5 :repeat-count 3)

They can also have warnings. A warning will get triggered for the days before the 25th and on the 25th itself.
